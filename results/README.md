# Results directory

This directory contains script-level derived outputs. Each subfolder corresponds to one experiment or diagnostic script. These outputs are included to preserve provenance.

The final manuscript-facing figures are available in `../figures/`.

The final manuscript-facing tables are available in `../tables/`.

Binary arrays such as `.npy` trajectory files and null-lifetime arrays are not included in this GitHub release. Large or binary generated data are stored separately or can be regenerated from the scripts.
